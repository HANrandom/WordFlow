# WordFlow 单词流 - 智能词汇管理工具

WordFlow 是一款极简、高效的单词管理与复习工具，采用 **React (Frontend)** + **Python (Backend)** + **SQLite (Database)** 构建，支持桌面端运行。

## 核心功能

### 1. 词库管理
- **手动添加**: 在主界面直接输入单词和释义，快速加入词库。
- **批量导入**: 支持 CSV 格式导入（第一列单词，第二列释义），一键迁移现有词库。
- **多数据库支持**: 点击顶部导航栏的数据库名称，可以**新建**或**切换**不同的 `.db` 文件，实现不同学科/语言的词库隔离。

### 2. 学习等级 (Levels)
- **自定义等级**: 点击顶部齿轮图标，可以重命名、添加或删除学习等级。
- **默认等级**:
  - 未筛选 (Level 0)
  - 记不太清 (Level 1)
  - 知中文不知英文 (Level 2)
  - 不会拼写 (Level 3)
  - 已熟练掌握 (Level 4)
- **快速筛选**: 在主页点击等级卡片，即可快速查看该等级下的所有单词。

### 3. 单词本 (Wordbook)
- **批量操作**: 进入“单词本”视图，支持多选单词进行**批量移动**或**批量删除**。
- **搜索**: 实时搜索词库中的单词。

### 4. 数据导出
- **CSV 导出**: 导出为标准的 CSV 文件，方便在 Excel 或其他工具中查看。
- **TXT 导出**: 导出为 Tab 分隔的文本文件，适合导入 Anki 等记忆软件。

## 运行指南

### 1. 开发环境 (Web 预览)
在 AI Studio 终端中运行：
```bash
npm run dev
```
访问预览窗口即可使用。

### 2. 桌面版运行 (本地电脑)
1. **安装依赖**:
   ```bash
   pip install pywebview
   npm install
   ```
2. **构建前端**:
   ```bash
   npm run build
   ```
3. **启动应用**:
   ```bash
   python main.py
   ```

## 数据存储
所有数据均存储在项目根目录下的 `.db` 文件中（默认为 `words.db`）。你可以随时备份这些文件以保护你的词库数据。

---

## 快捷键与技巧
- **回车键**: 在添加单词时，按回车键可快速提交。
- **实时同步**: 所有的修改都会实时保存到 SQLite 数据库中。
